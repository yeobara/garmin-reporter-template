# 가민 데이터, 매일 AI가 분석해서 이메일로 보내준다 (무료, 코드 몰라도 됨)

달리고 나서 가민 앱 열어보면 숫자가 잔뜩 나오죠.

페이스, 심박수, 케이던스, 지면 접촉 시간, 수직 진동, VO2max...

수면도 마찬가지예요. 수면 점수 72점? 깊은수면 1시간 12분? **이게 뭘 의미하는 건지** 앱이 설명을 안 해줘요.

그래서 직접 만들었습니다.

## 준비물이 이게 전부입니다

- 가민 기기 (수면 측정 되는 모델)
- GitHub 계정 (없으면 무료 가입)
- Gemini API 키 (구글 계정으로 2분 안에 발급)
- Resend 계정 (이메일 발송용, 무료)

**비용: 0원. 코딩: 없음. 설정 시간: 30분.**

---

## 설정 방법

### 1. 템플릿 복사

아래 링크에서 **[Use this template]** 초록 버튼 클릭
→ 내 GitHub 계정으로 복사됩니다

👉 **https://github.com/yeobara/garmin-reporter-template**

### 2. API 키 2개 발급

**Gemini API 키** (AI 분석용)
1. aistudio.google.com 접속
2. [Get API key] → [Create API key] 클릭
3. 키 복사 (`AIza...` 로 시작)

**Resend API 키** (이메일 발송용)
1. resend.com 접속 → 회원가입
2. [API Keys] → [Create API Key]
3. 키 복사 (`re_...` 로 시작)

### 3. GitHub에 값 5개 입력

내 레포 → Settings → Secrets and variables → Actions → New repository secret

| 이름 | 입력할 값 |
|------|---------|
| GARMIN_EMAIL | 가민 로그인 이메일 |
| GARMIN_PASSWORD | 가민 비밀번호 |
| GEMINI_API_KEY | 방금 복사한 키 |
| RESEND_API_KEY | 방금 복사한 키 |
| REPORT_TO | 리포트 받을 이메일 |

### 4. Actions 탭에서 활성화

레포 → Actions 탭 → [enable them] 클릭

끝입니다. **다음 날 오전 9시에 첫 이메일이 옵니다.**

## 주의사항 하나

가민 계정에 **2단계 인증(MFA)이 켜져 있으면 동작하지 않아요.**

가민 Connect 앱 → 프로필 → 계정 설정 → 보안 → 2단계 인증 해제 후 사용하세요.
